-----------------------------------------------------------------------------------------------
Maintainer      :       penghb
-----------------------------------------------------------------------------------------------
Howto           :       dhcp-server.sh
-----------------------------------------------------------------------------------------------
Evaluation      :       Setup a network alias. Then start the dhcpd server on this interface.
			Check the output contains the expected strings.
-----------------------------------------------------------------------------------------------
Notes           :       There should only be a maximum of 2 IPs associated with hostname
			i.e. 'hostname -i' should not output more than 2 IPs.
			If this is not the case, modify the /etc/hosts file accordingly,
			and then run the tests
