Crypt::SSLeay - OpenSSL support for LWP

This Perl module provides support for the HTTPS protocol under LWP,
to allow an LWP::UserAgent object to perform GET, HEAD and POST requests.

where LWP - The World-Wide Web library for Perl
set of Perl modules which provides a simple and
consistent application programming interface (API) to the World-Wide Web.
