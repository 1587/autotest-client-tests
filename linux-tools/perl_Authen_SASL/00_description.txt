Authen::SASL - SASL Authentication framework

SASL is a generic mechanism for authentication used by several network
protocols.
Authen::SASL provides an implementation framework that all protocols should be
able to share. 

Usage:
To run all tests:

perl-Authen-SASL		perl-Authen-SASL.sh

To run test individually:
cd /opt/fiv/ltp/testcases/fivextra/perl-Authen-SASL
perl t/<test.t>
